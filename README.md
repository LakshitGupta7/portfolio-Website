# Portfolio Website 🚀

A modern, interactive portfolio website built with **React**, **TypeScript**, and **Three.js**, featuring smooth **GSAP** animations and an **AI-powered chatbot**. And yes, **it's vibe coded** ✨.

## 🔗 Live Demo
[Check out the live app here!](https://lakshitgupta-portfoliowebsite.vercel.app/)

## ✨ Key Features
- **3D Hero Section:** Engaging landing page with a 3D avatar and immersive WebGL elements.
- **Project Showcase:** A slider displaying my best work with detailed descriptions and images.
- **AI Chatbot (Nexus):** An intelligent assistant powered by Google Gemini to answer questions about my background and skills.
- **Interactive UI:** Smooth transitions and micro-interactions using GSAP.
- **Responsive Design:** Fully optimized for all screen sizes.
- **Tech Stack Overview:** A dedicated section highlighting my technical expertise.
- **Professional Career Timeline:** Visual representation of my work experience.

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Vite
- **Animations:** GSAP (GreenSock Animation Platform)
- **3D Graphics:** Three.js, @react-three/fiber, @react-three/drei
- **AI Integration:** Google Generative AI (Gemini Flash)
- **Styling:** Vanilla CSS
- **Analytics:** Vercel Analytics

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/LakshitGupta7/portfolio-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-Website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```

### Build
Build the project for production:
```bash
npm run build
```
